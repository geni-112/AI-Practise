# Environment and live resource map

This skill operates the DockOne OBS -> MRS Iceberg -> DataArts -> DWS demo. It must not persist Huawei Cloud passwords, AK/SK, SSH private keys, or Terraform state. Read credentials from the active shell, a local encrypted store, or a cloud secret service.

## Required environment variables

Set these before running live cloud scripts:

```powershell
$env:HUAWEICLOUD_ACCESS_KEY = "<from secure store>"
$env:HUAWEICLOUD_SECRET_KEY = "<from secure store>"
$env:HUAWEICLOUD_SECURITY_TOKEN = "<optional temporary token>"
$env:HUAWEICLOUD_REGION = "la-south-2"
$env:HUAWEICLOUD_PROJECT_ID = "09d63c269e80f5e32f4ec00754ed462d"
$env:DEPLOYMENT_OBS_BUCKET = "hwstaff-retail-lakehouse-09d63c-20260622"
$env:DEPLOYMENT_MRS_CLUSTER_ID = "2a45044c-7506-4b96-9993-70d8a8397ed5"
$env:DATAARTS_WORKSPACE_ID = "c4fc507387fa4d2983d569c490525f86"
$env:DWS_HOST = "182.160.26.143"
$env:DWS_PORT = "8000"
$env:DWS_DATABASE = "gaussdb"
$env:DWS_USER = "dbaadmin"
$env:DWS_PASSWORD = "<from secure store>"
$env:ICEBERG_RUNTIME_JAR = "<local path to iceberg-spark-runtime jar>"
```

`ICEBERG_RUNTIME_JAR` can be replaced with `ICEBERG_RUNTIME_JAR_NAME` only when the runtime JAR is already uploaded to OBS under `jobs/`.

## Streaming extension variables

Set these only when running the optional RDS for PostgreSQL -> DMS Kafka -> MRS Flink -> OBS raw JSON ingress:

```powershell
$env:RDS_PGHOST = "<rds-postgresql-host>"
$env:RDS_PGPORT = "5432"
$env:RDS_PGDATABASE = "<database>"
$env:RDS_PGUSER = "<user>"
$env:RDS_PGPASSWORD = "<from secure store>"
$env:RDS_PGSSLMODE = "prefer"

$env:DMS_KAFKA_BOOTSTRAP_SERVERS = "<broker1:9092,broker2:9092>"
$env:DMS_KAFKA_TOPIC = "dockone.billing.contracts"
$env:DMS_KAFKA_GROUP_ID = "mrs-flink-contracts-to-obs"
$env:DMS_KAFKA_SECURITY_PROTOCOL = "<PLAINTEXT-or-SASL_SSL>"
$env:DMS_KAFKA_SASL_MECHANISM = "<optional>"
$env:DMS_KAFKA_USERNAME = "<optional>"

```

The rendered Flink SQL should not contain a real Kafka password in a committed or packaged file. Use the placeholder generated by `scripts/streaming/render_contracts_flink_sql.py`, then inject the password only in the secure job submission/runtime path.

## DPAPI bootstrap pattern on Windows

If the local encrypted credential file exists, load it without printing secrets:

```powershell
$credPath = Join-Path $env:LOCALAPPDATA "Codex\huawei-cloud-bigdata\credentials.xml"
$c = Import-Clixml $credPath
function Convert-SecureStringToPlainText([securestring]$s) {
  $bstr = [Runtime.InteropServices.Marshal]::SecureStringToBSTR($s)
  try { [Runtime.InteropServices.Marshal]::PtrToStringBSTR($bstr) }
  finally { [Runtime.InteropServices.Marshal]::ZeroFreeBSTR($bstr) }
}
$env:HUAWEICLOUD_ACCESS_KEY = Convert-SecureStringToPlainText $c.AccessKey
$env:HUAWEICLOUD_SECRET_KEY = Convert-SecureStringToPlainText $c.SecretKey
```

Use the same pattern for DWS password if it is stored locally. Never echo the resulting values.

## Resource naming conventions

- Cloud resources that can be renamed in place should use the `dockone` or `dockone-iceberg` prefix.
- Raw OBS files are JSONL/manifest data, not Iceberg.
- MRS converts raw JSONL into Apache Iceberg tables.
- Golden data is published by MRS/DataArts and loaded into DWS for BI or third-party consumption.
- Read `references/architecture_current_state.md` before asserting that the live resources fully match the latest S3/OMS and RDS/DMS/Flink two-flow architecture.
